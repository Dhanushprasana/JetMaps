Admin LIVE

The live dashboard makes use of Unity 3D engine, that was created based on the exact replica of karnataka's busy roads and streets.

Simulated 3D Admin dashboard for monitoring city traffic congestion and hotspots. SOS alert system within the admin dashboard is to be used in the event of emergencies on the streets

Admin Forecast

The forecasting dashboard, estimates possible currect state in the next 5 minutes.

This offers the system to be efficient in catching the congestion and traffic flow, based on the expected time frame and balanced ratios of the current state and forecasting state, then further notifies the Jetmaps troubleshoot mobile appliaction to resolve the congestion.

Analytical dashboard

The Analytical dashboard offers the visualised data from which the insights could be drawn easily for constructing and managing roads in the future.

Analyses regular flow of vehicles and identifies the key congestion causing factors that disrupts the flow based on Time Series Analysis (TSA).
